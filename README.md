# Soil-reports-apps

Code for three versions of the soil sampling report apps.
1. Basic app that requires uploading both soil sampling observational data and lab data to produce soil reports.
2. This version takes soil sampling data from the database, and only needs lab results uploading.
3. This version takes both soil sampling observational data and lab results from the database.

4. Also, report for generating FCC report from JSON according to Advisor requested specifications
